## Keybindings and Settings Location

The `keybindings.json` and `settings.json` files are located in the following directory:

```plaintext
~/.config/Code/User/
```

This is where you can customize your Visual Studio Code settings and keybindings to enhance your coding experience.

## Installing Extensions

To install vs code extensions from your list, run the following command:

```bash
xargs -L 1 code --install-extension < /path/to/your/extensions-list.txt
```
